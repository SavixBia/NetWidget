# NetWidget ☕

**Minimal & elegant network widget for Windows 10/11**

A tiny always-on-top widget to view and change your IP in seconds — no bloat, no ads.

### Features
- Real-time IP display
- One-click Static IP ↔ DHCP switch
- Unlimited saved network profiles
- Quick ping tool
- Modern dark Fluent Design with smooth animations
- System tray icon + double-click show/hide
- Remembers position and starts with Windows
