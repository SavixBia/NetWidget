Thank you for using NetWidget!

What it does
• Shows your current IP in real time
• Switch between Static IP and DHCP with one click
• Save and load unlimited network profiles (home, office, client…)
• Quick ping tool
• Modern dark Fluent Design with smooth animations
• System tray icon + double-click to show/hide
• Remembers window position and starts with Windows

How to use
1. Run SETUP.exe (requires administrator rights the first time)
2. Drag the widget anywhere on the screen
3. Click the arrow button to expand/collapse
